# SUS Digital - Plataforma Educacional

## Sobre o Projeto

Plataforma educacional para capacitação em saúde digital, desenvolvida pelo Núcleo de Saúde Digital (NSDIGI).

## Como executar o projeto

### Pré-requisitos

- Node.js & npm instalados - [instalar com nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

### Passos para execução

```sh
# Passo 1: Clone o repositório
git clone <URL_DO_REPOSITORIO>

# Passo 2: Navegue para o diretório do projeto
cd sus-digital-course-craft-main

# Passo 3: Instale as dependências
npm install

# Passo 4: Inicie o servidor de desenvolvimento
npm run dev
```

## Tecnologias utilizadas

- shadcn-ui
